# Quick Start

Add Rusticx to your rust proj

## Installation

Add rusticx in your Cargo.toml file

### Using cargo cli
<Tabs items={['cargo']} defaultIndex={1}>
  <Tabs.Tab>
  ```rust
  cargo add rusticx
  ```
  </Tabs.Tab>
</Tabs>

### Manual Installation

Add this line in Cargo.toml

```toml
rusticx = { version = '<version>', features = ['postgres']}
```

- use postgres feature for postgres
- use sqlite feature for SQLite
- use mysql feature for MYSQL
